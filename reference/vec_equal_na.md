# Missing values

**\[deprecated\]**

`vec_equal_na()` has been renamed to
[`vec_detect_missing()`](https://vctrs.r-lib.org/reference/missing.md)
and is deprecated as of vctrs 0.5.0.

## Usage

``` r
vec_equal_na(x)
```

## Arguments

- x:

  A vector

## Value

A logical vector the same size as `x`.
