# 64 bit integers

A `integer64` is a 64 bits integer vector, implemented in the `bit64`
package.

## Usage

``` r
# S3 method for class 'integer64'
vec_ptype_full(x, ...)

# S3 method for class 'integer64'
vec_ptype_abbr(x, ...)

# S3 method for class 'integer64'
vec_ptype2(x, y, ...)

# S3 method for class 'integer64'
vec_cast(x, to, ...)
```

## Details

These functions help the `integer64` class from `bit64` in to the vctrs
type system by providing coercion functions and casting functions.
